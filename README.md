# CSV to NFT with Node.js and React.js

User can upload CSV file to mint NFT, and store in IPFS.


## Backend
- npm install
- npm start

create a public directory in backend root

## Front end
- npm install
- npm start

## Smart contract

deploy on rinkeby test network after develope smart contract
